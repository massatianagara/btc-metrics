BTC metrics, your way.

A lightweight WordPress plugin that outputs text‑only shortcodes for Bitcoin price, NAV, gains, yield (YTD/QTD/Annual/Locked), and volatility (HV/IV). Style everything with your own theme.

Unduh Plugin Dukung Pengembangan
Open source BTC‑only No CSS lock‑in

Kenapa Satia BTC Metrics?

Ringkas & fleksibel

Simple shortcodes with sensible defaults. Choose currency format, decimals, millions/full scale, and live auto‑refresh.

Sumber data nyata

Price from CoinGecko (Binance fallback). Implied volatility from Deribit with blended HV estimate when IV is unavailable.

Yield yang praktis

Compute YTD/QTD, annual (Jan→Dec), annual_ytd (Jan→today), and annual_locked (Dec 31 of chosen year → today).

Update Bar

Timezone‑aware “Updated Xm ago” text with data‑source labels. Works anywhere you put a live widget.

Tangkapan Layar Plugin

Klik gambar untuk memperbesar.

Shortcode Cepat

Price & NAV

[sbm_tracker metric="price"]
[sbm_tracker metric="nav" scale="millions"]

Gains

[sbm_tracker metric="gain_btc" type="baseline"]
[sbm_tracker metric="gain_usd" type="ytd" scale="full"]

Yield

[sbm_tracker metric="yield" type="ytd"]
[sbm_tracker metric="yield" type="annual" year="2024"]
[sbm_tracker metric="yield" type="annual_locked" year="2024"]
[sbm_tracker metric="yield" type="annual_ytd" year="2024"]

Volatility & Update Bar

[sbm_tracker metric="histvol" window="30"]
[sbm_tracker metric="implvol"]
[sbm_update_bar style="full" tz="Asia/Jakarta"]

Instalasi

  1. Download the latest release from GitHub.
  2. In WordPress: Plugins → Add New → Upload Plugin. Upload the ZIP and activate.
  3. Open BTC Metrics → Settings to set currency, decimals, timezone, holdings, and baseline.
  4. Insert shortcodes in any post, page, or block (supports classic & block editor).

FAQ

Theme compatibility

Output is text-only so it inherits your theme or page builder styling. No CSS lock-in.

Data sources

Price: CoinGecko (Binance fallback). IV: Deribit. HV uses daily log returns with a blended fallback when IV is unavailable.

Yield types

YTD, QTD, annual, annual_ytd, and annual_locked (choose the year).

Instalasiation

Download the ZIP from GitHub, upload via Plugins → Add New → Upload Plugin, then activate.
